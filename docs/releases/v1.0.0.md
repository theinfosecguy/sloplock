# SlopLock v1.0.0

SlopLock is now positioned as a GitHub Action dependency gate for pull requests.

## Install

```yaml
name: SlopLock

on:
  pull_request:

permissions:
  contents: read
  pull-requests: write

jobs:
  sloplock:
    name: SlopLock dependency gate
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - uses: theinfosecguy/sloplock@v1
```

## Highlights

- Blocks newly introduced dependency names that do not exist in public registries.
- Blocks packages first published inside the configured cooldown window.
- Scans pull request dependency changes by default.
- Supports npm, PyPI, Go, crates.io, Maven Central, NuGet.org, Packagist, and RubyGems.org.
- Reports through annotations, job summaries, and optional sticky pull request comments.
- Works in read-only repositories when comments are disabled.

## Notes

SlopLock is intentionally narrow. It is not an SCA scanner, vulnerability
scanner, typosquat detector, install-script analyzer, or package reputation
score.
